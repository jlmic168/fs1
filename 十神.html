十神

生我者: 同性为偏印(卩) / 異性为正印(印)

同我者: 同性为比肩(比) / 異性为劫财(劫)

我生者: 同性为食神(食) / 異性为伤官(伤)

我克者: 同性为偏财(偏) / 異性为正财(财)

克我者: 同性为七杀(杀) / 異性为正官(官)


**男命 (六亲关系)**

偏印 / 生我者+同性 / 祖父,舅,岳父,养母

正印 / 生我者+異性 / 母亲,姨,长辈,贵人,师长

比肩 / 同我者+同性 / 兄弟,堂兄弟,姨表兄弟,姑父,朋友,同辈

劫财 / 同我者+異性 / 姐妹,堂姐妹,姨表姐妹,朋友,同辈

食神 / 我生者+同性 / 男孙,外公,外甥,晚辈,学生,部属

伤官 / 我生者+異性 / 孙女,祖母,岳母,外甥女,晚辈,学生,部属

偏财 / 我克者+同性 / 父亲,伯叔,舅表兄弟,小舅,小妾

正财 / 我克者+異性 / 妻子,姑,舅表姐妹,大小姨子

七杀 / 克我者+同性 / 儿子,侄子,姑表兄弟,姐夫妹夫

正官 / 克我者+異性 / 女儿,外婆,姑表姐妹,侄女,长官,上司,师长


**女命 (六亲关系) **

偏印 / 生我者+同性 / 母,姨,孙女

正印 / 生我者+異性 / 祖父,舅,孙子,女婿,长辈,贵人,师长

比肩 / 同我者+同性 / 姐妹,堂姐妹,姨表姐妹,朋友,同辈

劫财 / 同我者+異性 / 兄弟,堂兄弟,姨表兄弟,家公,朋友,同辈

食神 / 我生者+同性 / 女儿,祖母,外甥女,晚辈,学生,部属

伤官 / 我生者+異性 / 儿子,外甥,外公,晚辈,学生,部属

偏财 / 我克者+同性 / 家姑,姑,舅表姐妹,外孙女,养父,

正财 / 我克者+異性 / 父亲,伯叔,舅表兄弟,外孙

七杀 / 克我者+同性 / 外婆,儿媳,姑表姐妹,大小姑子,情夫,

正官 / 克我者+異性 / 丈夫,姐夫妹夫,姑表兄弟,小叔,长官,上司,师长
